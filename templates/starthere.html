How to Run ComplyAI

✅ Best: Python HTTP Server

python -m http.server 8000

Then open: http://localhost:8000

✅ Alternative: VS Code Live Server

Install "Live Server" extension → Right-click HTML file → Open with Live Server

❌ Don't: Open Files Directly

Do NOT double-click HTML files or use file:// protocol. Navigation and localStorage won't work!
